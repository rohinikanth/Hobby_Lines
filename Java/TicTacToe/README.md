#  Tic Tac Toe – Java Desktop Application

A complete Tic Tac Toe game built using **Java Swing** with a clean UI, sound effects, and an intelligent computer opponent.

---

##  Features

-  Interactive 3×3 grid UI (Java Swing)
-  Computer opponent logic
-  Sound effects for moves, wins, and draws
-  Live scoreboard (Player / Computer / Draws)
-  Restart game functionality
-  Clean separation of UI, logic, and sound

---

## Tech Stack

- **Java**
- **Swing (GUI)**
- **OOP principles**
- **Audio (javax.sound.sampled)**
